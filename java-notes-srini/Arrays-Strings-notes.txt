Array
=======

Array is a container which is used to store the collection of data elements which is with the same datatype and fixed in size.


syntax
------

// Array declaration

datatype []variableName;

datatype variableName [];

datatype[] variableName;

int[] a;


//Array initialization

datatype[] variableName = new datatype[size];


int []a= new int[3]

a[0]=10;
a[1]=20;
a[2]=30;


int[] a={1,2,3,4,5,6,7};

Note: Array size represents how many values we are store into an Array.

-> Array will be stored data into based on size and Array index will always starts from Zero(0).


simple example
--------------

public class ArrayDemo {

	public static void main(String[] args) {	
		
		int [] arr=new int[3];

	}
}

write a java program to print MIN element and MAX element of an array
===================
package com.sriniit.java.arraystrings;

public class ArrayElementsMinMax {

	public static void main(String[] args) {

		int[] arr = { 10, 6, 15, 22, 2 };

		int min = arr[0];  //10
		int max = arr[0];  //10, 6

		for (int i = 0; i < arr.length; i++) {
			if (arr[i] > max) {

				max = arr[i];
			}			
			
			if (arr[i] < min) {

				min = arr[i];
			}
		}

		System.out.println("max element is : " + max);
		System.out.println("min element is : " + min);
	}
}

Strings
===========

-->String is used to store group of characters

	String str="Srini";

--> String is a predefined class which is available in java.lang package
--> we can use String as a data type(referenced data type)

note : Every java class can be used as a referenced data type

--> String is immutable in java(we can't modify the data once created)


-->  We can create the String object in 2 ways

		1) String str= "Srini";  //string literal ---here data is going save in SCP(string constant pool)

		2) String str1= new String("Srini-it-solution");   // here data is going to save in heap memory 


immutability
==============
--->immutability means once we created the object, its state(data) can not be modified
--> In java, String is immutable that means if we try to modify string object, it will create new object with modified data


why String made as immutable
===============================

String are used in Security-sensitive areas like File paths, DB urls and username, password and class loading, if Strings are mutable, There is a possibility of 
change the DB Urls which is security risk, this is one of the reason to made String as immutable.

String Constant pool
=======================

SCP is a special memory area in JVM heap memory to store the string objects

	String str1="Srini";
	
	String str2="Srini";

If str1 and str2 objects are having same content, only one object will be created and two variable references will be pointed to that object.


-> If we create String objects using "new" keyword, always object will be created inside the Heap memory.


public class StringClassDemo {

	public static void main(String[] args) {		
		
		String str1="Srini";
		
		String str2="Srini";    // in SCP object will be created
		
		System.out.println(str1==str2); 
		
		
		
		String str3= new String("Srini=it");  // in Heap memory object will be created		
		
		String str4= new String("Srini=it");		
		
		
		System.out.println(str3==str4);  // false

			System.out.println(str3.equals(str4));  // true
		
	}
}

String class methods
=========================

chatAt()  ==> To get the character based on given index

		String str="srini-it";
		System.out.println(str.charAt(5));

length() ==> To get the size of given String(no. of characters in a String)

		System.out.println(str.length());

concat() ==> To join two Strings

			String str="srini-it-";		
			String str1="solutions";

			System.out.println(str.concat(str1));

equals()  ==> It is used to compare the content of two Strings

		String str="srini-it-";		
		String str1="solutions";

		System.out.println(str.equals(str1));  // false

note : In Strings == operator will compare the address and equals() method will compare the content. 


replace() ==> To replace the existing characters with another in a give String

		String str="srini-it-";
		System.out.println(str.replace("srini-it-", "srini-it-solutions"));

toUpperCase()  ==> To convert the String into uppercase

			str.toUpperCase();

toLowerCase()  ==> To convert the String into lowercase

			str.toLowerCase();

indexOf()   ==> To get the first occurrence of the character 

		String str="srinivas battineni";		
		System.out.println(str.indexOf('t'));

lastIndexOf() ==> to get the last occurrence of the character

		String str="srinivas battineni";
		
		System.out.println("First Index =="+ str.indexOf('i'));   //2		
		System.out.println("Last Index ==="+str.lastIndexOf('i')); //17

note : If the given character is not available it will return -1
